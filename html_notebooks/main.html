Handwriting prediction¶

Using the mnist dataset we are going to create a model that can predict handwritten numbers. The training and testing data will come from the mnist dataset with thousands of handwritten numbers.

We will be creating a model, train, test and validate it using our own numbers to check how good the model is. You can find all the handwritten images inside the '/images' folder. I have handwritten 20 numbers. The first 10 are drawn as normal as I could. The next 10 images are each number written with a few variations, such as a very big 0 or a thin 8 ... to see if the model can classify these right.

# This model is created using 200 epochs. If you don't want to wait for around 1h, then just execute the cell where the already trained model is saved and loaded.

# creating the model
model = tf.keras.models.Sequential()

# adding the layers (input, hidden, output)

# will take the 28x28 and make one list of 784 pixels
model.add(tf.keras.layers.Flatten(input_shape=(28,28)))

# add the hidden layers with the relu activation
model.add(tf.keras.layers.Dense(256, activation='relu'))
model.add(tf.keras.layers.Dropout(0.2))
model.add(tf.keras.layers.Dense(256, activation='relu'))
model.add(tf.keras.layers.Dropout(0.2))
model.add(tf.keras.layers.Dense(256, activation='relu'))
model.add(tf.keras.layers.Dropout(0.2))
model.add(tf.keras.layers.Dense(256, activation='relu'))
model.add(tf.keras.layers.Dropout(0.2))
model.add(tf.keras.layers.Dense(256, activation='relu'))
model.add(tf.keras.layers.Dropout(0.2))

# add an output layer of 10 choices (0-9). Softmax makes sure that all the output values add to 1.
model.add(tf.keras.layers.Dense(10, activation='softmax'))

# compile the model
opt = tf.keras.optimizers.Adam(learning_rate=0.001)
model.compile(optimizer=opt, loss='sparse_categorical_crossentropy', metrics=['accuracy'])

# fitting the model with the training data using 200 epochs together with the img augmentation

fitted_model = model.fit(
    datagen.flow(X_train, y_train, batch_size=128),
    validation_data=(X_test, y_test),
    epochs=200
)

img_num = 0

while os.path.isfile(f"images/image-{str(img_num)}.png"):
	try:
		# read the img, but only give the black and white colors
		img = cv2.imread(f"images/image-{str(img_num)}.png")[:,:,0]
        
		# inverse the img, because images are drawn in white on black and not black on white
		img = np.invert(np.array([img]))
        
		# predict the img using argmax (get the neuron with the highest validation)
		prediction = model.predict(img)
		print(f"You have written a: {np.argmax(prediction)}")
		
		# show the img on screen
		plt.imshow(img[0], cmap=plt.cm.binary)
		plt.show()
	except:
		print("Something went wrong")
	finally:

		# go to the next file
		img_num += 1
